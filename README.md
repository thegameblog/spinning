Running locally
---------------

1. Make sure [Gesso][] is installed

   ```bash
   $ npm install -g gesso
   ```

2. Install dependencies

   ```bash
   $ npm install
   ```

3. Run the server

   ```bash
   $ gesso serve
   ```

Now visit [http://localhost:65330](http://localhost:65330/).
